# langy

Text language detection witth ISO 639-1

## Install

```
$ npm i @yoannchb/langy
```

## How to use ?

```js
import langy from 'langy';

const result = langy(
  'Hello everyone I hope you had a great day ! Dont forget to do your homework after school !'
);
console.log(result);
```
